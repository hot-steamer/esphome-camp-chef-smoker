This repo retro-fits wifi capabilities to monitor a non-wifi camp-chef smoker from home assistant.  Some resistors and wires will need to be added if you copy me directly.  If you solder the two UART wires to the resistor pads directly, you can skip the resistors.

This component is only meant to communicate with firmware version F1NW V1.0.8.  Smokers that come with WiFi will very likely have a different firmware with a different protocol.
